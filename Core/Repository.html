    class Core::Repository

    • Core::Repository
    • Reference
    • Object

    Overview

    Repository is a gateway between Models and Database.

    Supported methods:

    • #query (alias of #query_all)
    • #query_one
    • #query_one?
    • #insert
    • #update
    • #delete
    • #exec
    • #scalar

    See Query for a handy queries builder.

    logger = Core::Logger::IO.new(STDOUT)
repo = Core::Repository.new(db, logger)

user = User.new(name: "Foo")
user = repo.insert(user)
# INSERT INTO users (name, created_at) VALUES ($1, $2) RETURNING *
# 1.773ms

query = User.last
user = repo.query_one(query)
# SELECT * FROM users ORDER BY id DESC LIMIT 1
# 275Î¼s

user.name = "Bar"
repo.update(user)
# UPDATE users SET name = $1 WHERE (id = $2) RETURNING *
# 1.578ms

repo.delete(user)
# DELETE FROM users WHERE id = $1
# 1.628ms
